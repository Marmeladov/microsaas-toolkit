# 🚀 MicroSaaS Toolkit
**Ship your SaaS in record time.**

MicroSaaS Toolkit is a modular, enterprise-grade foundation for .NET developers. It eliminates the "boilerplate phase" of building a SaaS by providing pre-built, decoupled modules for the most common requirements.

## ✨ Features (Included in the Toolkit)

Based on a clean architecture and modular design, the toolkit includes:

* **Multi-Tenancy:** Secure data isolation and tenant resolution.
* **Billing & Subscriptions:** Out-of-the-box support for **Stripe** and **LemonSqueezy**.
* **Usage Tracking:** Track API calls and resource consumption for metered billing.
* **Onboarding:** Streamlined user and organization setup flows.
* **EFCore Persistence:** Optimized database schemas for SQL Server, PostgreSQL, and more.
* **Abstractions:** Decoupled interfaces allowing you to swap providers easily.

---

## 🏗 Modular Structure

The toolkit is organized into logical components to keep your project clean:

- `MicroSaas.Toolkit.Abstractions`
- `MicroSaas.Toolkit.Billing` (Stripe, LemonSqueezy, EFCore)
- `MicroSaas.Toolkit.Tenancy` (Runtime, Abstractions)
- `MicroSaas.Toolkit.Usage` (API, Runtime)
- `MicroSaas.Toolkit.Onboarding`
